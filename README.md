# Wellai


## - Guide
### 1. git clone

```
git clone https://kdt-gitlab.elice.io/ai_track/class_03/ai_project/team10/wellai.git
```

### 2. git submodule 설정

```
(서브모듈 처음 사용시)
git submodule init
git submodule update
```

### 3. sub레포지토리 변경사항 업데이트
git에 반영된 변경사항들을 로컬로 불러옵니다.

```
git submodule update --remote
```

### 4. 업데이트 사항 git 반영
```
git add
git push
```
